# Changelog

All notable changes to AWS Cost Sentinel will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of AWS Cost Sentinel
- AWS Cost Explorer integration
- ML-based anomaly detection using Isolation Forest
- Multi-channel notifications (Slack, Discord, Email, SNS)
- Daily, weekly, and monthly cost summaries
- Budget threshold monitoring
- Service-level cost breakdown
- Cost spike detection
- AWS Lambda deployment support
- Docker containerization
- CLI interface
- Comprehensive documentation
- CI/CD pipeline with GitHub Actions

### Features
- Real-time cost monitoring
- Configurable sensitivity levels for anomaly detection
- Customizable budget thresholds
- Top services ranking
- Cost forecasting
- Historical trend analysis

### Added
- Initial public release
- Core monitoring functionality
- Basic notification system
- Lambda deployment scripts
- Documentation and setup guides

### Security
- Environment variable support for sensitive credentials

- Secure webhook handling
